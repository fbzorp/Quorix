# Quorix Messenger

Decentralized, end-to-end encrypted messaging app with voice & video calls and **instant finality** powered by Yellow Network.  
Building live as part of Yellow Acceleration Builders program (EthCC Cannes Demo Day goal).

## Status

- Phase: Idea alignment → MVP build
- Yellow integration: [In progress / Planned for state channels / ERC-7824 / instant settlement]
- Current focus: Simplistic UI + E2EE WebRTC calls + Yellow instant finality proofs

## Features (target)

- E2EE 1:1 and group text messaging
- E2EE voice & video calls (WebRTC-based)
- Instant finality for message delivery / payments / events via Yellow Network
- Very clean, intuitive UI (minimalist design)
