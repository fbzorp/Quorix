# Contributing to Quorix

Thanks for helping build a superior decentralized messenger!

1. Fork & branch from `main`
2. Work in feature branches (`feat/e2ee-calls`, `feat/yellow-finality`)
3. Open PRs with clear descriptions
4. Follow code style (TBD — add eslint/prettier later)
5. All code must be open-source compatible (MIT)
